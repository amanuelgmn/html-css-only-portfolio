# Portfolio with JavaScript Enhancements

This version of the portfolio includes JavaScript to add interactive features and improved user experience.

## Live Preview

View the live version at: [https://jsversion.netlify.app/](https://jsversion.netlify.app/)
## Added Features

- **Dark/Light Theme Toggle**: Switch between dark and light themes with persistent preference storage.
- **Smooth Scrolling Navigation**: Active navigation highlighting that updates based on scroll position.
- **Reveal Animations**: Elements animate into view as you scroll down the page.
- **Back to Top Button**: Appears when scrolling down for easy navigation to the top.
- **Interactive Contact Form**: Real-time validation with error messages for form fields.
- **Interactive Cards**: Hover effects and animations on project/service cards.
- **Filter Chips**: Interactive filtering for portfolio items.
- **Rotating Text**: Dynamic text rotation in the hero section.
- **Animated Statistics**: Counters that animate when the stats section comes into view.
- **Hero Glow Effects**: Dynamic glow animations in the hero section.
