# Amanuel Getachew Portfolio Website

A modern, responsive personal portfolio website built with plain HTML and CSS.  
It presents profile information, projects, technical skills, and contact options in a clean single-page layout.

## Live Demo

- [plainhtmlcssportfolio.netlify.app](https://plainhtmlcssportfolio.netlify.app)

## Project Highlights

- **Single-page portfolio** with clear sections: About, Projects, Skills, and Contact
- **Responsive design** optimized for desktop, tablet, and mobile screens
- **Dark/Light theme toggle** for improved usability and visual preference
- **Project showcase cards** for selected work and responsibilities
- **Tech stack chips** with icons for quick skill scanning
- **Contact form** with required inputs (`name`, `email`, `subject`, `message`)

## Tech Stack

- `HTML5`
- `CSS3`
- Google Fonts (`Inter`, `Space Grotesk`)

## File Structure

- `index.html` - Page structure and portfolio content
- `styles.css` - Theme, layout, responsive behavior, and component styling


## Main Sections

- **Hero**: Introduction and quick actions
- **About**: Personal engineering focus and interests
- **Projects**: Featured project cards
- **Skills**: Core technologies and capabilities
- **Contact**: Collaboration message, contact form, and GitHub profile link
