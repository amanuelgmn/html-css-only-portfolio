# Portfolio Website - Two Versions

This repository contains two versions of my personal portfolio website: one built with plain HTML and CSS, and an enhanced version with JavaScript for added interactivity.

## Versions

### HTML & CSS Only Version
A modern, responsive portfolio built with plain HTML and CSS. Features a clean single-page layout with sections for profile, projects, skills, and contact.

**Live Preview**: [https://plainhtmlcss.netlify.app/](https://plainhtmlcss.netlify.app/)

### JavaScript Enhanced Version
This version includes all the features of the HTML/CSS version plus JavaScript enhancements for improved user experience, including theme toggling, smooth scrolling, animations, and interactive elements.

**Live Preview**: [https://jsversion.netlify.app/](https://jsversion.netlify.app/)

## Project Structure

- `html_css_only/` - Plain HTML and CSS version
- `html_css_js/` - JavaScript enhanced version
